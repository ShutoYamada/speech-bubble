# プロジェクト概要

- 翻訳と吹き出しによる可視化を行うプロダクトのMVP
- アプリケーション上でカメラおよびストレージから動画を読み込み、動画中の人物の口の開閉を検知する
- 口の開閉検知と動画から得られた音声によって動画中の発言した人物を特定する
- 音声から翻訳されたテキストを生成し、人物に対して吹き出しをオーバレイさせる
- 翻訳元となる言語は「日本語以外」とし、翻訳先の言語は「日本語」とする

# 基本構成

- バックエンドはTypescript + Hono、クライアントはReact + viteとする
- それぞれ `apps/api`と`apps/web`で管理する
